# D&D Skills

This is a repo containing Agent Skills that can be used for Dungeons and Dragons.

It it intended as a collection of skills that can be used by dungeon masters to help them generate campaigns using tools such as Claude Code, Cursor, or Copilot.

## Planned Skills

The following skills are planned to be included in this repo.

- determine-encounter-difficulty
- find-creature
- create-encounter
- create-dungeon
