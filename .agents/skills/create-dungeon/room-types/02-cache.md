# The Cache

## Description
A hidden or protected location where items of value are stored.

## Purpose
- To reward exploration.
- To provide useful tools or consumables for the rest of the dungeon.
- To hint at the owners or former inhabitants of the dungeon.

## Components
- **Barrier:** A lock, a hidden door, a minor trap, or a puzzle. (Use the `create-puzzle` skill).
- **Loot:** Gold, silver, gems, potions, scrolls, or minor magic items. (Use the `create-treasure` skill).
- **Theme:** The items should be logically placed (e.g., an armory has weapons, a kitchen has spices/food).
